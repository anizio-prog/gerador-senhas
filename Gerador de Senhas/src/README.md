# Gerador de Senhas

Projeto simples e funcional de um gerador de senhas personalizável, feito com HTML, CSS e JavaScript.

![Exemplo do Gerador](./screenshot.png)

---

## Funcionalidades

- Define a quantidade de caracteres
- Inclusăo opcional de letras maiúsculas, minúsculas, números e símbolos
- Geraçăo de senhas seguras e aleatórias
- Interface amigável e responsiva

---

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Selecione as opçőes desejadas.
3. Clique no botăo "Gerar Senha".
4. Copie a senha gerada e use onde precisar!

---

## Tecnologias usadas

- HTML5
- CSS3
- JavaScript (ES6)
